# Sheets Grant Tracker

Reads your grant tracker daily and posts what's due in 30, 60, and 90 days — and which drafts need attention now.

## Prerequisites
- A Google account with access to the grant tracker spreadsheet (you'll OAuth into Google Sheets)
- A Slack workspace where you can install the agent's bot and invite it to one or more channels
